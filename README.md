## ğŸš€ Getting Started

### Prerequisites
- Java JDK 8 or higher
- Maven 3.6+
- IDE (IntelliJ IDEA / Eclipse)
- Chrome/Firefox browser

### Installation

1. Clone the repository
```bash
git clone https://github.com/yashyp12/Selenium-Java.git
cd Selenium-Java
```

2. Install dependencies
```bash
mvn clean install
```

3. Run tests
```bash
mvn test
```
